# PRIZM BRAND — LEGACY DROP

Responsywna landing page dla marki **PRIZM BRAND** i pierwszego oficjalnego dropu **LEGACY**.

Strona prezentuje limitowaną koszulkę **Legacy Tee**, historię marki, loty, informacje o zakupie, tabelę rozmiarów oraz prosty chat pomocniczy działający na słowach kluczowych.

## Podgląd

Projekt działa jako klasyczna strona internetowa napisana w:

- HTML
- CSS
- JavaScript
- PHP API

Nie używa frameworków frontendowych.

## Funkcje strony

- tryb jasny i ciemny,
- responsywny wygląd na telefonie i komputerze,
- sekcja hero z modelem i przyciskiem CTA,
- slider w sekcji genezy marki,
- pozioma karuzela lotów,
- sekcja produktu z ceną i opisem,
- modal z tabelą rozmiarów,
- system polubień produktu,
- licznik wyświetleń,
- przycisk audio,
- animacje podczas scrollowania,
- prosty PRIZM AI Chat bez zewnętrznego API,
- możliwość rysowania po stronie za pomocą canvas.
